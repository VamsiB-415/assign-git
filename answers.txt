• What does fast-forward in git mean?
Ans:We use fast-forward when we want to reach the target branch from the current branch tip 
by shifting the pointer forward from current branch tip to target branch.

• Why should you have different branches?
Ans:We need different branches so that many developers can work on same project simultanously with out any conflicts and errors.
Each branch has same instance of the entire code that developers can work on with out disturbing the actual code.
